# Emotion AI System

This project is an AI-based system that:
- Detects user emotion from text
- Predicts intensity of emotion
- Suggests actions based on emotion and context

## How it works
1. Text is cleaned using preprocessing
2. Emotion is predicted using ML model
3. Intensity is predicted using additional features
4. Decision engine suggests actions

## Run the project
pip install pandas numpy scikit-learn

python main.py
